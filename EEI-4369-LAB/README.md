## 🚀 Features Implemented

### ✅ Lab 01 – Basic Android UI
- Created a new Android Studio project.
- Used Java as the programming language.
- Simple GUI built using ConstraintLayout.

### ✅ Lab 02 – Multimedia Integration
- Added images/videos to the UI.
- Used background media to enhance the visual experience.

### ✅ Lab 03 – Google Maps Integration
- Integrated Google Maps API.
- Geocoding support to display address location on a map.

### ✅ Lab 04 – Sensor Usage
- Accessed temperature sensor via emulator.
- Played a local audio clip when the temperature exceeded a threshold (last two digits of SID).

### ✅ Lab 05 – SQLite Database Integration
- Connected multimedia UI to SQLite.
- Stored user input.
- Displayed a toast message on successful insert.
